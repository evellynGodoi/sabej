Sobre o Projeto

O SABEJ é um projeto desenvolvido pelos discentes do curso de Engenharia de Computação da Universidade São Francisco, por meio da componente curricular Circuitos Elétricos.

O objetivo principal do projeto é aplicar conceitos teóricos em uma prática funcional, criando protótipos que integram eletrônica, programação e controle automatizado.

Um dos protótipos desenvolvidos no âmbito do SABEJ é o sistema de irrigação automática inteligente, que combina sensores, atuadores e comunicação com serviços online para otimizar a irrigação de plantas de maneira econômica e eficiente.

O projeto permite demonstrar, de forma prática, como tecnologias de baixo custo podem ser aplicadas para resolver problemas cotidianos, como o consumo excessivo de água ou a falta de monitoramento constante de plantas.

Componentes Utilizados

O protótipo de irrigação automática desenvolvido pelo projeto SABEJ utiliza uma combinação de hardware simples, porém eficiente, permitindo a demonstração prática de conceitos de eletrônica, programação e automação.

Microcontrolador ESP8266 NodeMCU V3

O ESP8266 NodeMCU V3 é o cérebro do protótipo. Ele é um microcontrolador com conectividade Wi-Fi integrada, permitindo que o sistema se conecte à internet para obter dados de previsão do tempo através da API OpenWeatherMap.

Além disso, ele lê os sinais do sensor de umidade do solo e controla a ativação da bomba de água por meio do módulo de relé. Sua função é processar informações do sensor e da API, decidir quando e quanto irrigar, e acionar a bomba de forma automática.

Sensor de Umidade Capacitivo

O sensor capacitivo de umidade do solo mede a quantidade de água presente no solo. Diferente dos sensores resistivos, ele não oxida facilmente, garantindo maior durabilidade.

O sensor envia uma leitura analógica para o ESP8266, que interpreta se o solo está seco, úmido ou molhado. Sua função é fornecer dados em tempo real sobre a umidade do solo, permitindo que o sistema decida se a irrigação é necessária ou não.

Mini Bomba Água Submersa 6V

A bomba de água submersa é responsável por transportar a água do reservatório até o solo da planta. Ela funciona com baixa tensão (6V) e possui vazão adequada (120L/h) para irrigar uma planta de forma controlada, evitando desperdício.

Sua função é executar a irrigação física da planta conforme as instruções recebidas pelo ESP8266 via relé.

Módulo de Relé 1 Canal 5V

O módulo de relé funciona como um interruptor controlado eletronicamente. Ele permite que o microcontrolador ESP8266, que opera com sinais de baixa tensão, ligue ou desligue a bomba de água, que requer maior corrente.

O LED integrado indica quando o relé está ativo, facilitando a visualização do funcionamento do sistema. Sua função é controlar a bomba de água com segurança, isolando o circuito de alta potência do microcontrolador.

Protoboard 400 Pontos

A protoboard é utilizada como base para organizar a montagem do protótipo sem a necessidade de solda. Ela permite conectar o ESP8266, o sensor de umidade e o módulo de relé de forma clara e segura, facilitando ajustes e testes durante a demonstração.

Sua função no projeto é servir como plataforma de montagem e conexão dos componentes, mantendo o protótipo organizado e funcional.

Como Funciona

Lógica de Funcionamento

  1. O ESP8266 se conecta ao Wi-Fi
  2. Consulta a API OpenWeatherMap para obter previsão do tempo
  3. Lê o sensor de umidade do solo
  4. Analisa as condições climáticas e a umidade
  5. Decide automaticamente quando e quanto irrigar
  6. Aciona a bomba através do módulo de relé

API de Previsão do Tempo

O sistema utiliza a OpenWeatherMap API para obter dados climáticos em tempo real:

  • Temperatura atual
  • Umidade do ar
  • Condição do céu (ensolarado, nublado, etc.)
  • Previsão de chuva
  • Velocidade do vento

A API gratuita permite até 1.000 chamadas por dia, ideal para projetos educacionais.
